# Intelligent Portfolio

An AI-powered portfolio platform built with Next.js and a Python FastAPI backend.
Features dynamic LangGraph agents to personalize the landing page for different visitors based on their roles.

## Architecture
- **Frontend**: Next.js, React, Tailwind CSS, Zustand
- **Backend**: Python, FastAPI, LangGraph, Qdrant (Vector DB)
- **CI/CD**: GCP Cloud Build automatically deploys to Cloud Run
